# Handrail

Safe execution + deterministic replay for AI-generated code.

## Install

curl -s https://handrail.dev/install.sh | bash

## Usage

handrail execute plan.json
handrail doctor

## Why Handrail

- Deterministic: 1000/1000 runs verified identical
- Governed: All operations through quorum + 6 gates
- Provable: Immutable ledger of every execution
- Replayable: Reproduce any execution exactly
